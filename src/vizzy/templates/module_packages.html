How to find what adds these

These packages are added by NixOS modules you've enabled. To find which module adds a specific package:

  1. Search your config for the package name: grep -r "orca" ~/nixos-config
  2. Check related services: grep -r "gnome\|accessibility" ~/nixos-config
  3. Look in nixpkgs for which module defines it: search nixos/modules/ on GitHub
